## Recording Guidelines

See [../RECORDING_GUIDE.md](../RECORDING_GUIDE.md) for detailed setup instructions.

### Quick Checklist

- [ ] Terminal: Dark theme, JetBrains Mono 14pt, 120x30
- [ ] Browser: Clean profile, dark mode, 110% zoom
- [ ] Screen: 1920x1080 or 2560x1440
- [ ] Audio: 48kHz, noise gate enabled
- [ ] Environment: Do Not Disturb enabled

### Key Principles

1. **Show, don't tell** - Demonstrate actions visually
2. **Keep it focused** - One concept per tutorial
3. **Pause for effect** - Let viewers read output
4. **End with next steps** - Guide to the next tutorial
