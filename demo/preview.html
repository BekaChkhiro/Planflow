Demo Production Checklist

  • Review DEMO_SCRIPT.md storyboard
  • Set up terminal (font, theme, size)
  • Practice terminal flow from TERMINAL_SESSION.md
  • Record terminal scenes
  • Record browser/dashboard scenes
  • Generate GIF using hero.tape or manual recording
  • Edit full demo video
  • Export hero GIF (under 5MB)
  • Export full demo (MP4, 1080p)
  • Create social media clips
  • Upload to hosting/CDN

Quick Commands

# Install VHS (terminal GIF generator) brew install vhs # Generate hero GIF from tape script vhs hero.tape # Convert SVG to PNG (requires Inkscape) inkscape hero-animation.svg --export-filename=hero.png --export-width=1600 # Optimize GIF gifsicle -O3 --lossy=80 hero.gif -o hero-optimized.gif
